# What are variables?
Variables are the data that you will be manipulating when you do any programming. They can be created and edited by you. Every variable has a type, whether you can see it or not and this dictates what you can do with the variable.

```python
name = "Burger"
amount = 7
cheesy = True
```
# Variable types

| Type       | Aliases         | What it is                                                                   |
| ---------- | --------------- | ---------------------------------------------------------------------------- |
| String     | `str`           | Text                                                                         |
| Integer    | `int`,`i8`      | Numbers which are not decimal                                                |
| Float      | `double`, `f64` | Numbers which are decimal                                                    |
| Array      | `Vec`, `List`   | A list of another type of variable                                           |
| Dictionary | `HashMap`       | A selection of key value pairs where u can find the values by inputting keys |

> [!NOTE]
> # Pointers
These are variables which **point** to other variables. These may seem incredibly **point**less at first until you realise that otherwise the computer would have no idea where your variables where. These generally only show up in low level languages as they interact very directly with the system.
