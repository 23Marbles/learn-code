# What is programming?
At its core programming is simply the process of manipulating data automatically. Whether you are making the bare metal of an operating system in C or a simple Excel automation script in Python all your code really does is reading and writing data stored somewhere on your computer. Even actions like drawing to the screen is just changing some data in a very specific place. Almost all of the time you will not actually be spelling out where and how data is stored manually and your language if choice will do it for you, but it is still useful to understand that this is all your code is doing.
# Levels of language 
Languages can generally be classified as high or low level languages which show how closely you interact with the machine. On general high level languages are easier to learn but slower and low level is faster but harder to learn.
